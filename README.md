# Zynk Admin Panel

Management dashboard for Zynk Quick Commerce platform.

## Features
- Real-time analytics dashboard
- Product & inventory management
- Order management & tracking
- Driver management & onboarding
- User management
- Pricing & promotional offers
- Sales reports & analytics

## Tech Stack
- Next.js 14
- Supabase (Backend & Auth)
- Tailwind CSS
- Recharts (Analytics)

## Setup
```bash
npm install
npm run dev
```
